# webpack-boilerplate-leemir

my js webpack boilerplate

## Installation

### Environment

> OS : Windows10 x64
>
> Node.js : v14.17.3(LTS)
>
> npm : v6.14.13

### Dependencies

```shell
npm install
```

- [x] babel
- [x] bundling css
  - [x] bundling scss(sass)
- [ ] bundling img
- [ ] bundling font
- [ ] tree shaking
- [x] webpack split(mode)
