# webpack-boilerplate-leemir

my js webpack boilerplate

## Installation

### Environment

> OS : Windows10 x64
>
> Node.js : v14.17.3(LTS)
>
> npm : v6.14.13
>
> webpack : v5.x
>
> babel : v7.x

### Dependencies

```shell
npm install
```

### Run

```shell
npm start # dev
npm run build # prod
```

## Feature

- [x] babel
- [x] bundling css
  - [x] bundling scss(sass)
- [x] bundling img
  - [x] base64 encoding(size < 4kb)
- [x] bundling font
- [x] tree shaking
- [x] webpack split(mode)
